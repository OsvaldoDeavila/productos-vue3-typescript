# Catálogo de Productos - Vue 3

Aplicación web tipo e-commerce que permite listar, buscar, filtrar y visualizar el detalle de productos.

## Características

- Listado de productos
- Búsqueda por nombre
- Filtro por categoría
- Paginación
- Vista de detalle de producto

---

## Tecnologías utilizadas

- Vue 3 (Composition API)
- Vue Router
- TypeScript
- Tailwind CSS
- Vite

## Arquitectura

El proyecto está basado en **Clean Architecture**, separando responsabilidades en capas:

### 📌 Principios aplicados

- Separación de responsabilidades
- Bajo acoplamiento
- Reutilización de lógica
- Escalabilidad del proyecto

## 🧠 Metodología

Se aplicó una arquitectura basada en:

- 🧩 Clean Architecture
- 🔄 Separación por capas (Domain, Infrastructure, Presentation)
- 📦 Uso de casos de uso (Use Cases)
- 🧾 Tipado fuerte con TypeScript

## 📦 Instalación

Clona el proyecto:

```bash
git clone https://github.com/OsvaldoDeavila/productos-vue3-typescript/tree/main/src/infrastructure/api
cd productos-vue3-typescript
```

Instala dependencias:
```bash
npm install
```

### A tener en cuenta 

##### Variables de entorno

Crear un archivo .env en la raíz:

VITE_API_URL=https://fakestoreapi.com/products


Arrancar proyecto
```bash
npm run dev
```

Abrir en el navegador:

http://localhost:5173
